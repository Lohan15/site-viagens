# 🌍 ExploreMundo - Agência de Viagens

O **ExploreMundo** é um projeto de site institucional para uma agência de viagens, focado em proporcionar uma experiência visual atraente e organizada para usuários que buscam destinos turísticos. O projeto conta com uma interface responsiva, formulário de contato funcional e integração visual com mapas e ícones modernos.

---

## 🚀 Funcionalidades

* **Página Inicial (Hero):** Destaque visual com chamada para ação (CTA).
* **Galeria de Destinos:** Exibição de cards interativos com efeito *hover* e layout em grid (3 colunas).
* **Seção Sobre:** Informações institucionais sobre a agência.
* **Página de Todos os Destinos:** Listagem completa de locais com filtragem visual.
* **Formulário de Contato:** Interface amigável para envio de mensagens.
* **Informações de Localização:** Integração visual de mapa e dados de contato com ícones.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **Frontend:** HTML5 e CSS3 (Flexbox e Grid Layout).
* **Ícones:** [Font Awesome](https://fontawesome.com/) e [Google Material Icons](https://fonts.google.com/icons).
* **Tipografia:** Arial e fontes nativas sans-serif.
* **Backend (Em desenvolvimento):** Planejado para integração com **Java** e **SQL** para gerenciamento de destinos e mensagens de contato.

---

## 🎨 Layout e Design

O design foi pensado para ser limpo e profissional, utilizando uma paleta de cores baseada em tons de azul e laranja:
* **Azul Principal:** `#2e7db5` (Confiança e Tecnologia)
* **Laranja de Destaque:** `#e8760a` (Energia e Ação)
* **Fundo:** `#f0eeee` (Leveza e Modernidade)
